## 汉化
汉化部分：
1 汉化语言包。
2 汉化全部前台内嵌英文。
3 汉化后台内嵌设置选项标题，汉化部分内嵌设置选项描述。
4 优化中文语言显示支持。
5 部分不影响使用的后台内嵌英文没有汉化，如需深度汉化请联系汉化作者。
